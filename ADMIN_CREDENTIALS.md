## Security Notes

⚠️ **Important Security Measures:**
- This file should NEVER be committed to version control
- Admin credentials should be rotated regularly in production
- Enable two-factor authentication for admin accounts
- Monitor admin activity logs
- Restrict admin access to authorized personnel only
- Use strong, unique passwords in production environment

## Demo Mode Information

In the current demo/development environment:
- Any email containing "admin" grants admin access
- Passwords are not validated (for demo purposes only)
- In production, implement proper authentication with:
  - Strong password requirements
  - JWT token validation
  - Rate limiting
  - Session management
  - Activity logging
