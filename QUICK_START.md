## 🎯 How to Use

### 1️⃣ Start the Application
The development server should already be running. If not, check the preview panel.

### 2️⃣ Test Light/Dark Mode
- Look for the **sun/moon icon** in the top-right navbar
- Click it to toggle between light and dark themes
- The preference is saved automatically!

### 3️⃣ Login as User
1. Click **"Sign In"** or **"Login"** button
2. Enter: `user@airport.com` + any password
3. You'll see the **User Dashboard**

**What you can do as a User:**
- Browse and search lounges
- View your bookings
- See membership status
- Generate QR codes
- View reward points
- Access your profile

### 4️⃣ Login as Admin
1. Logout (click your name → Logout)
2. Login with: `admin@airport.com` + any password
3. Click **"Admin Portal"** in the navbar (golden/amber colored)

**What you can do as Admin:**
- **User Management**: View, edit, delete all users
- **Lounge Management**: Add, edit, delete lounges
- **Booking Management**: View, approve, cancel bookings
- **Membership Management**: Create, edit membership plans
- **Analytics**: View revenue, user growth, trends
- **Notifications**: Send system-wide notifications
- **System Settings**: Database, access control, security
- **Everything Else**: Full system control!
